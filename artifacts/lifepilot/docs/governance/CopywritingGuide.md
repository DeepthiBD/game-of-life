## Part 1 — LifePilot Voice Guide

### The Voice in One Sentence

> **LifePilot speaks like the most encouraging person the child has ever met — the one who always believes in them, never judges them, and always reminds them their future is full of possibility.**

### What LifePilot sounds like

| Attribute | In practice |
|-----------|------------|
| **Encouraging** | Celebrates every step, not just the destination |
| **Friendly** | Warm and conversational — like a trusted guide, not a textbook |
| **Hopeful** | Always pointing toward a positive future |
| **Curious** | Asks questions that make the child think, not answer "correctly" |
| **Supportive** | Present when things are hard, not just when things are going well |
| **Positive** | Reframes challenges as growth opportunities |
| **Playful** | Light and fun — the adventure is enjoyable |
| **Adventurous** | Every session is a new leg of a journey |
| **Inspiring** | Connects today's small effort to a meaningful future |
| **Reflective** | Creates space for honest self-examination without pressure |
| **Confident** | Assured without being arrogant — "you've got this" energy |

### What LifePilot does NOT sound like

| Category | Example of what to never write |
|----------|-------------------------------|
| **Corporate** | "Please complete the onboarding process." |
| **Academic** | "Describe your daily learning outcomes in detail." |
| **Administrative** | "Action required: profile incomplete." |
| **Productivity** | "You are 23% efficient this week." |
| **Judgemental** | "You haven't finished your task." |
| **Clinical** | "Input your mood rating on a scale of 1 to 10." |
| **Pressuring** | "Don't lose your streak!" |
| **Comparing** | "Others your age have completed 3 missions." |

### The Brand Voice Position

```
LifePilot is:                     LifePilot is NOT:
─────────────────────────         ─────────────────────────
A trusted guide            ≠      A teacher
A supportive co-pilot      ≠      A parent replacement
A future-focused mentor    ≠      A therapist
A positive companion       ≠      A supervisor or authority
```

## Part 2 — Tone Matrix

The tone shifts slightly by context — the core voice stays constant, but the energy adapts.

| Context | Energy | Keywords | Example |
|---------|--------|----------|---------|
| **Cockpit (home)** | Warm, welcoming, excited | adventure, ready, today | "Welcome back, Captain! Ready for today's mission?" |
| **New session** | Energetic, curious | explore, discover, journey | "A new day, a new leg of your journey. What's next?" |
| **Completing something** | Celebratory, warm | great, progress, proud | "Another step on your journey. Future You will appreciate this!" |
| **Reflection prompt** | Gentle, curious, safe | wonder, notice, feel | "What surprised you today?" |
| **Empty state** | Inviting, open, unhurried | waiting, whenever, ready | "Your Flight Log is ready whenever you are." |
| **Error** | Calm, reassuring, forward-looking | try again, safe, continue | "Something got a little turbulent. Your progress is safe — let's try again." |
| **Future Me** | Magical, warm, timeless | future, horizon, waiting | "A message from Future You is waiting for you." |
| **Career Explorer** | Curious, expansive, possibility | explore, world, discover | "The Technology World is full of possibilities. Where do you want to start?" |
| **Money Quest** | Playful, practical, empowering | choose, imagine, quest | "You have ₹500. What would you do?" |
| **Life Choices** | Thoughtful, non-judgemental | consider, choose, explore | "There's no wrong answer here. What feels right to you?" |
| **Parent (CoPilot)** | Respectful, warm, family-first | family, conversations, together | "Here's what [Name] has been exploring this week." |
| **School** | Professional, future-ready, supportive | skills, growth, readiness | "Your students are building life skills this week." |
| **Notification** | Gentle, helpful, optional | ready, waiting, whenever | "Your journey is waiting whenever you're ready." |
| **Achievement** | Joyful, proud, growth-focused | earned, growth, milestone | "You've unlocked a new Explorer Badge! ✨" |

### Tone that is always constant (regardless of context)

- Never judgmental
- Never pressuring
- Never comparison-based
- Never shame-inducing
- Always future-oriented
- Always growth-mindset

## Part 4 — Microcopy Standards

Microcopy is the small text that surrounds interactions — labels, placeholders, helper text, confirmations. It has enormous impact on how the product feels.

### Principle: Every word earns its place

Microcopy should be:
- **Minimal** — if it can be cut, cut it
- **Encouraging** — even placeholder text should feel welcoming
- **In LifePilot language** — no generic defaults

### Button copy

| Context | ✅ Use | ❌ Never use |
|---------|--------|-------------|
| Starting something new | Explore / Start Mission / Begin / Launch | Submit / Execute / Proceed |
| Continuing something | Continue / Keep going / Next step | Next / Continue (acceptable) / Proceed |
| Saving | Save / Keep this | Submit / Confirm / OK |
| Completing | Land it! / Accomplished! / Done! | Submit / Complete / Finish |
| Sending Future Me letter | Send to the future / Launch | Submit / Send / OK |
| Logging a reflection | Land it / Log this moment | Submit / Save / Post |
| Cancelling | Not now / Maybe later | Cancel / Dismiss / No |
| Deleting | Remove this | Delete / Erase |
| Going back | Back / Return | Back |

### Placeholder text

| Field | ✅ Use | ❌ Never use |
|-------|--------|-------------|
| Reflection text area | "What's on your mind today?" | "Enter your reflection here." |
| Goal / Flight Plan name | "My destination is..." | "Enter goal title" |
| Future Me letter | "Dear Future Me, right now I am..." | "Enter message text" |
| Name field (onboarding) | "My name is..." | "Enter name" |
| City field | "My city is..." | "Enter city" |

### Helper text / field labels

| Field | ✅ Use | ❌ Never use |
|-------|--------|-------------|
| Target date | "When do you want to arrive?" | "Target date" |
| Goal category | "What kind of adventure?" | "Category" |
| Delivery date (Future Me) | "When should Future You read this?" | "Delivery date" |
| Why this goal | "Why does this matter to you?" | "Description" |

### Confirmation copy

| Action | ✅ Use | ❌ Never use |
|--------|--------|-------------|
| Goal created | "Flight plan launched! ✈️" | "Goal saved successfully." |
| Reflection logged | "Logged! ✨" | "Entry saved." |
| Letter sent | "Sent to the future! ✨" | "Message submitted." |
| Badge earned | "New Explorer Badge unlocked! 🏅" | "Achievement unlocked." |
| Habit logged | "Done for today! ✅" | "Habit marked complete." |
| Goal completed | "You've landed! 🎉" | "Goal completed." |
| Mission done | "Mission accomplished! 🎯" | "Task completed." |

## Part 5 — Notification Standards

### Notification principles (from XP-005 + XP-003)

1. Notifications are **opt-in** — never opt-out
2. Maximum **1 per day** per child
3. Delivered in the **morning only** (7am–12pm, user-configured time)
4. Must feel **helpful, not pressuring**
5. Never reference streaks in a pressuring way
6. Always include an easy path to dismiss or disable

### ✅ Approved notification templates

**Daily journey reminder**
```
"Good morning! Your journey is waiting whenever you're ready. ✈️"
"A new day, a new adventure. What will you discover today?"
"Ready to explore something new today?"
```

**Future Me letter ready**
```
"A message from Future You is waiting to be opened. ✨"
"Future You left you a note. Ready to read it?"
```

**New mission available**
```
"A new adventure path is ready to explore."
"Ready for a quick mission today?"
```

**Streak celebration (milestone only)**
```
"You've been on a 7-day journey! ✈️ Keep flying!"
"30 days of showing up — that's something to be proud of! 🌟"
```

**Return after absence**
```
"Welcome back! Your journey is right where you left it."
"Great to see you again. Ready to continue?"
```

### ❌ Never send these notifications
```
"Don't break your streak! Log in now."
"Your streak ends at midnight!"
"You haven't opened the app in 3 days."
"Complete now — you are overdue."
"Urgent: action required."
"You are falling behind."
```

### Notification tone rules

| Rule | Standard |
|------|---------|
| Subject / opener | Always warm — "Good morning", "A message", "Ready for" |
| Streak mention | Only on milestone (7, 30, 100 days) — never as a threat |
| Urgency | Never — all notifications are gentle invitations |
| Personalisation | Use pilot name when available: "Good morning, Captain Priya!" |
| Emoji | One per notification, at the end — never multiple |
| Length | ≤ 80 characters for headline, ≤ 120 for body |

## Part 6 — Mission Writing Standards

Missions are the primary action units in LifePilot. Their copy sets the tone for the entire experience.

### Mission naming conventions

| Component | Standard | Example |
|-----------|---------|---------|
| Mission title | Action verb + adventure noun | "Explore the Technology World" |
| Mission description | 2–3 sentences, conversational, future-oriented | "Technology shapes the world you'll grow up in. This mission takes you inside the world of software, AI, and digital design. What could you build one day?" |
| Step labels | "Step 1: [action]" → "First, [action]" | "First, watch this short story about a software engineer." |
| Completion label | "Mission accomplished!" | Not "Task complete" |
| XP display | "+15 XP — Effort rewarded! ✨" | Not "15 points added" |

### Mission description formula

```
[Hook — why this matters to the child's future]
[What they'll do — in one sentence]
[Future connection — what this opens up]
```

Example:
> "The world runs on code — and you can be one of the people who builds it. In this mission, you'll meet real engineers and explore what they do every day. What might you create one day?"

### Mission difficulty labels

| Internal | Copy shown to child |
|---------|---------------------|
| easy | Starter mission |
| medium | Explorer mission |
| hard | Challenge mission |
| epic | Legend mission |

### Mission state labels

| State | Copy |
|-------|------|
| Not started | "Ready to explore" |
| In progress | "Your mission continues..." |
| Completed | "Mission accomplished! 🎯" |
| Locked (wave-gated) | "Coming soon on your journey" |

## Part 7 — Reflection Prompt Standards

Reflection prompts are the most personal copy in LifePilot. They must feel like a trusted friend asking — not a teacher assigning.

### The prompt principles

1. **One question only** — never a list, never compound
2. **Open-ended** — no yes/no, no correct answer
3. **Conversational** — short, warm, natural language
4. **Future-linking** — where possible, connect to Future You
5. **Shame-free** — every question is safe to answer any way
6. **Judgment-free** — no correct answer exists

### ✅ Approved prompt bank

**Daily check-in**
- "What surprised you today?"
- "What made you smile today?"
- "What are you looking forward to tomorrow?"
- "What made today worth remembering?"
- "How do you feel right now — in one word?"

**Growth**
- "What challenge helped you grow today?"
- "What did you discover about yourself today?"
- "What would you do differently tomorrow?"
- "What's something you tried for the first time recently?"
- "What's one small thing you're proud of today?"

**Future-linking**
- "What would Future You thank you for today?"
- "What is one step you took toward who you want to become?"
- "If Future You could send you a message right now, what might they say?"
- "What's one thing you're building — even slowly — toward your future?"

**Gratitude & relationships**
- "Who inspired you today?"
- "How did you show kindness today?"
- "Who would you like to thank today, and why?"
- "What's something or someone you're grateful for right now?"

**Curiosity**
- "What's something you want to learn more about?"
- "What question is stuck in your mind right now?"
- "What's something you saw or heard that made you think?"

### ❌ Prompts to never use

```
"Describe your day in 200 words."
"Complete the reflection entry below."
"Provide a detailed account of your learning today."
"Rate your performance from 1 to 10."
"List three things you accomplished."
"Evaluate your progress against your goals."
```

### Prompt rotation

- Prompts rotate daily — no child sees the same prompt two days in a row
- The prompt bank has ≥12 prompts per category (maintained in `FLIGHT_LOG_PROMPTS` constant)
- Prompt selection uses date-seeded randomisation — same prompt on the same date for a given child

## Part 8 — Future Me Writing Standards

Future Me is LifePilot's most emotionally powerful feature. Its copy is held to the highest standard.

### The Future Me principle

Every piece of copy in Future Me should feel like:
- A warm letter
- A message across time
- A gift to your future self
- A moment of connection between who you are and who you're becoming

### Feature naming

| Element | ✅ Use | ❌ Never use |
|---------|--------|-------------|
| Feature name | Future Me | Future Letters |
| Letter term | Message to Future You | Future Letter / Scheduled message |
| Reading experience | "A message from your past self" | "Future Letter #7" |
| Delivery date | "Future You will read this on [date]" | "Delivery date: 01/01/2027" |
| Letter ID | Never show | Never show |
| Send action | "Send to the future" | "Submit" / "Schedule" |
| Read action | "Open your message" / "Read your message" | "View" / "Open" |
| Sent state | "Sent to the future! ✨" | "Submitted successfully" |
| Locked (not yet delivery date) | "Future You isn't ready to read this yet..." | "Not yet delivered" |

### Letter editor copy

| Element | ✅ Use |
|---------|--------|
| Letter opening | "Dear Future [Name]," |
| Placeholder | "Right now I am... / I'm hoping that by the time you read this..." |
| Word count guidance | None — never constrain the letter |
| Delivery date prompt | "When should Future You read this?" |
| Time presets | "1 month from now", "3 months from now", "6 months from now", "1 year from now" |

### Post-send experience

After a letter is sent, the child sees a warm narrative moment — not a confirmation alert:

> **"Sent to the future! ✨"**  
> "Your message is on its way to Future You. They'll be so glad you wrote."

After a letter is read, the child sees:

> **"A message from your past self"**  
> "This was written by you on [date]. They believed in you then — and still do."

### Tone guidance for in-app Future Me copy

| Moment | Tone | Example |
|--------|------|---------|
| Empty state (no letters yet) | Inviting, magical | "Write your first message to Future You. They're waiting to hear from you." |
| Letter written, not yet read | Anticipatory | "Future You is waiting. The message arrives on [date]." |
| Letter ready to read | Warm, exciting | "A message from your past self is waiting... ✨" |
| After reading | Reflective, proud | "That was written by you on [date]. Look how far you've come." |

## Part 9 — Error Message Standards

Errors should feel like a gentle bump in the road — not a crash, not a judgement.

### The three error principles

1. **Never blame the child** — errors are the app's responsibility
2. **Always reassure** — "your progress is safe" where applicable
3. **Always offer a path forward** — "let's try again"

### Error message formula

```
[What happened — in neutral terms] + [Reassurance] + [Next step]
```

### ✅ Error copy by context

| Error context | ✅ Use | ❌ Never use |
|--------------|--------|-------------|
| Generic / unknown | "Something got a little turbulent. Let's try again." | "Error 500. Internal server error." |
| Save failed | "We couldn't save that right now. Your progress is safe — try again." | "Save operation failed." |
| Network offline | "You're offline right now. Your journey data is safe on this device." | "No internet connection." |
| Load failed | "Something got in the way of loading this. Let's try once more." | "Page failed to load." |
| Input invalid (form) | "That didn't quite work. Could you check and try again?" | "Invalid input." |
| Not found | "This path doesn't exist yet — back to Cockpit?" | "404: Page not found." |
| Feature unavailable | "This part of the journey isn't ready yet — coming soon!" | "Feature unavailable." |
| Session expired | "Your session ended. Log back in to continue your journey." | "Session expired. Please re-authenticate." |
| Sync failed | "We couldn't sync right now. Everything is safe on your device." | "Sync error. Data not transmitted." |

### Error icon / visual guidance

- Use a gentle turbulence / weather visual — not a red X or warning triangle
- Error colour: Sunrise Orange (warm) — not red (aggressive)
- Icon: cloud with a small lightning bolt — not a stop sign

## Part 10 — Success Message Standards

Success messages celebrate effort and growth — never talent, never comparison.

### The success message principles

1. **Celebrate effort**, not talent: "You worked hard" not "You're so smart"
2. **Reference growth**, not perfection: "Great progress" not "Perfect"
3. **Connect to future self** where possible: "Future You will appreciate this"
4. **Never compare to others**: "Your best" not "Better than most"
5. **Genuine, not hollow**: vary the messages — same message every time feels robotic

### ✅ Success copy bank

**Generic completion**
- "Great progress today! ✨"
- "Another step on your journey. 🌟"
- "Nice work exploring! Keep going."
- "Future You will appreciate this. ✈️"
- "You showed up — that's what matters most."

**Flight Plan (goal) created**
- "Flight plan launched! ✈️"
- "Your destination is set. The journey begins now."

**Flight Plan (goal) completed**
- "You've landed! 🎉 What a journey."
- "Flight plan complete! Future You is proud."

**Reflection logged**
- "Logged! ✨ One more moment captured."
- "Another reflection on your journey. Well done."

**Mission accomplished**
- "Mission accomplished! 🎯"
- "You did it. Explorer points earned!"

**Badge earned**
- "New Explorer Badge unlocked! 🏅"
- "You've earned a Growth Moment. ✨"

**Habit streak milestone**
- "7 days of showing up! You're building something real. 🔥"
- "30-day journey! That's dedication. 🌟"

**Level up**
- "Level up! You're now a {{rank}}. 🚀"
- "New pilot rank unlocked: {{rank}}. The skies are yours! ✈️"

**XP earned**
- "+{{xp}} XP — Effort rewarded! ✨"
- "{{xp}} XP for exploring. Keep going!"

### ❌ Success phrases to never use

```
"Perfect!"
"You are the best!"
"You beat everyone."
"100% — top score!"
"You are a genius."
"You're so talented."
"Better than your classmates."
"First place!"
```

## Part 11 — Parent Communication Standards

### The parent tone

Parents are co-pilots — informed, respected, family-first. The tone with parents is:
- More professional than child copy
- Still warm and family-oriented (not clinical)
- Never fear-based or guilt-inducing
- Focused on conversations and connection, not surveillance

### Parent copy guidelines

| Context | ✅ Use | ❌ Never use |
|---------|--------|-------------|
| CoPilot welcome | "Here's what [Name] has been exploring this week." | "Activity report for [Name]." |
| Conversation starter | "Ask [Name] what they discovered in Career Explorer today!" | "[Name] completed 3 tasks." |
| Child away from app | "It's been a quiet week for [Name]. A gentle nudge might help." | "[Name] has not logged in for 7 days." |
| Feature explanation | "Flight Log gives [Name] space to reflect on their day." | "Journal module activated." |
| Privacy note | "[Name]'s reflections are private — they've chosen not to share these." | "No reflection data available." |
| Achievement | "[Name] reached a milestone this week! A great moment to celebrate together." | "[Name] completed 5 tasks." |

### Parent notification copy

```
✅ "[Name] wrote a message to Future [Name] this week. Ask them about it!"
✅ "[Name] explored the Technology World. Ask what they discovered."
✅ "A great week for [Name]'s journey. They might love to share what they found."
❌ "[Name] only completed 2 of 5 tasks this week."
❌ "[Name]'s activity is lower than average."
❌ "Action required: [Name] has not used the app."
```

### What parents never see (reinforce in copy)

If a parent tries to access private content:
> **"[Name] has chosen to keep their Flight Log private."**  
> "That's their space — and it's safe. Ask them what's on their mind today."

## Part 12 — School Communication Standards

### The school tone

School-facing copy is the most professional in the product. It respects educators' time and positions LifePilot as a trusted partner in student development — not a competitor to school or a toy.

| Attribute | Standard |
|-----------|---------|
| Tone | Professional, warm, future-ready |
| Vocabulary | "Students" (not "pilots" in school reports), "Educator", "Class" |
| Focus | Growth, life skills, wellbeing — never scores or rankings |
| Language | Clear, jargon-free, accessible to non-technical educators |

### School copy guidelines

| Context | ✅ Use | ❌ Never use |
|---------|--------|-------------|
| Class summary | "Class 8B explored 6 Career Worlds this week." | "Class 8B completion rate: 34%." |
| Student growth | "Students are building reflection habits." | "Students scored below average on tasks." |
| Module description | "Career Explorer helps students discover future possibilities." | "Career module tracks occupational preferences." |
| Parent report | "Encourage your students to share their journey with family." | "Student performance metrics available." |
| Onboarding | "Welcome to LifePilot for Schools. Your students' journeys start here." | "School admin portal initialised." |

### What school reports never include

- Individual student rankings
- Comparison between students
- Any child's private content (reflections, journals, letters)
- Time-in-app metrics (we don't position this as engagement)
- Score or grade-like outputs

## Part 13 — Copy Review Checklist

Every piece of copy — UI, notification, prompt, error, success, parent, school — must pass this checklist before shipping.

### Core compliance (must pass — zero exceptions)

| # | Check | Gate |
|---|-------|------|
| 1 | Uses XP-003 terminology (Flight Plan, not Goal; Pilot, not User) | ❌ = Rewrite |
| 2 | Encouraging tone — at least one positive element | ❌ = Rewrite |
| 3 | Hopeful tone — at least one forward-looking element | ❌ = Rewrite |
| 4 | Child-safe — age-appropriate for 10–16 | ❌ = Rewrite |
| 5 | No shame language | ❌ = Rewrite |
| 6 | No comparison language | ❌ = Rewrite |
| 7 | No pressure language | ❌ = Rewrite |
| 8 | Future-oriented — where relevant | ❌ = Rewrite |
| 9 | Adventure-oriented — in XP-003 spirit | ❌ = Rewrite |
| 10 | Growth mindset — effort praised, not talent | ❌ = Rewrite |

### Tone check (use this for nuanced decisions)

Read the copy aloud. Then ask:

1. If a 10-year-old read this, would they feel encouraged or judged?
2. If a 16-year-old read this, would they feel respected or talked down to?
3. If a parent saw this, would they trust LifePilot more or less?
4. Does this sound like a trusted guide or an authority figure?
5. Is there any word that could make the child feel bad about themselves?

If any answer is unfavourable → rewrite.

### Quick-scan prohibited words

If any of these words appear in child-facing copy → mandatory rewrite:

```
failed · incomplete · wrong · incorrect · behind · overdue · late
error · invalid · rejected · denied · locked · missed · required
rank · leaderboard · score · grade · performance · assessment · test
pressure · deadline · obligation · must (imperative) · should (judgemental)
perfect · best (comparative) · beat (competitive) · top (ranking)
```

- [ ] Copy exists in `en.json` (not hardcoded)

- [ ] Interpolation variables documented (`{{name}}`, `{{count}}`)
- [ ] Locale files for hi, ta, te, kn, ml, mr, bn, gu, pa are updated or flagged for translation
